# Victor AI 面试助手 - 产品概述

## 1. 产品简介

Victor AI 面试助手是一款面向求职者的智能面试练习平台，通过AI技术模拟真实面试场景，帮助用户提升面试能力。系统支持技术面试与行为面试，不限专业和行业，用户可自定义知识库、简历、岗位要求，实现个性化面试训练。

## 2. 产品定位

### 2.1 目标用户
- **主要用户**：求职者，用于面试练习和能力提升
- **适用场景**：校招、社招、转行求职、内部晋升面试等

### 2.2 核心价值
- **真实模拟**：语音对话模拟真实面试，支持实时字幕、追问、提示
- **个性定制**：用户自定义知识库、简历、岗位要求，匹配个人求职方向
- **多维评估**：多Agent评估团队，从多个维度独立评分，提供详细改进建议
- **数据驱动**：面试题关联原始文档，参考答案有据可依

### 2.3 产品形态
- **开源项目**，支持本地部署
- **前后端分离**架构
- **PC Web端**优先，移动端预留扩展

## 3. 核心功能概览

| 功能模块 | 核心能力 |
|---------|---------|
| 用户体系 | 注册登录、个人中心 |
| 资料系统 | 题库、岗位库、简历库、经历库 |
| 面试流程 | 模板选择、流程配置、多轮面试 |
| 面试交互 | 语音/文字模式、实时字幕、代码编辑、架构图绘制 |
| 面试评估 | 多Agent评估、多维度评分、追问机制 |
| 面试报告 | 详细报告、PDF/图片导出 |
| Agent管理 | 自定义Agent、Agent团队、模型配置 |
| 开放接入 | 开放API Key、外部数据导入、待审核资料 |

## 4. 产品边界

### 4.1 当前版本（V1.0）范围
- PC Web端应用
- 中文界面（国际化接口预留）
- 本地部署为主
- 简单注册登录
- 数据脱敏保护

### 4.2 当前版本不包含
- 移动端适配
- 面试预约/定时提醒
- 付费会员体系
- 多租户/SaaS模式

## 5. 术语定义

| 术语 | 定义 |
|-----|------|
| ASR | Automatic Speech Recognition，自动语音识别 |
| TTS | Text-to-Speech，文本转语音 |
| Agent | 智能代理，具有自主决策能力的AI实体 |
| Loop Agent | 迭代搜索代理，循环执行直到满足终止条件 |
| pgvector | PostgreSQL向量扩展 |
